## 團隊協作

- 從設計師的Figma中檢視與提取素材
- 使用版控git與後端工程師協作

## 前端開發

- 根據設計稿使用HTML, CSS, JavaScript進行切版、RWD、互動效果製作
- 動態效果使用GSAP製作
- 使用Gulp進行nunjuck(pug)和SASS的編譯，結合browser-sync即時預覽
- Vue3框架的使用，著重在傳統多文件結構的開發和可重用性的組件製作
- 資料串接搭配Axios

## 後端開發

- 使用PHP+MySQL製作後台管理系統，對資料的查詢、新增、修改、刪除等功能製作
